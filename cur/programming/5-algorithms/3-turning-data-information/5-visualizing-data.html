Agrupando los datos

DAT-2.E.3 solamente clasificando

La clasificación de datos es extraer grupos de datos con una característica en común.

Otra cosa que se hace con frecuencia en la ciencia de los datos es agrupar (o clasificar) los datos. Por ejemplo, aquí están los datos de los autos agrupados por la marca de vehículo (column a12):


  • La columna A muestra todas las marcas de vehículos (el campo 12 de cada registro).
  • La columna B muestra el número total de vehículos de cada marca.
  • La columna C contiene una lista de todos los datos de los autos para los vehículos de esa marca (por ejemplo, todos los datos de los Acuras o todos los datos delos Nissans). Si das doble clic en una de las listas de la columna C, se abrirá otra tabla que mostrará los datos de todos los autos de esa marca.

Agrupar es una función de orden superior así como mantener, mapear, combinar, y encontrar primero porque toma una función  (aquí, campo) como entrada.

  1. Abre tu proyecto U3L4-procesamiento-de-datos, si aún no lo tienes abierto.
  2. Determina una pregunta que puedas responder al agrupar tus datos, y crea el código para responderla.
    Da clic aquí para ver ejemplo de preguntas que requieren ser agrupadas para responder.
    • ¿Cuántos autos Toyota hay en la base de datos?
    • ¿Cuántos de los autos son híbridos en la base de datos?
    • ¿Cuántos autos Hyundai modelo hay en la base de datos? (Es necesario buscar dentro de una de las listas de la columna C, necesitarás dos funciones de mantener.)

    Pipe puede ser útil para las preguntas que requieren buscar dentro de las listas internas de los datos agrupados (en columna C).

La función de histograma trabaja como la función agrupar, excepto que permite especificar los límites superiores e inferiores de los datos que estás agrupando por el tamaño de las agrupaciones, y clasifica los datos por agrupación. Por ejemplo, aquí están los datos de los autos agrupados por MPG (columna 1):
  • Ahora, columna A muestra MPG (el primer campo de cada registro) agrupados en intervalos de 5 y clasificados.
  • Así como antes, column B muestra el número total de autos dentro de cada rango de MPG (0–5, 5–10, 10–15, etc.)
  • Así como antes, column C contiene una lista de todos los datos de los autos dentro de ese rango de MPG (por ejemplo, todos los datos de los 268 autos que obtienen entre 20 y 25 MPG en la ciudad).

Histograma es una función de orden superior así como mantener, mapear, combinar, encontrar primero, y agrupar porque toma la función (aquí, ambos campo y +5) como entrada.
